# 三世代構造の幾何学的起源 (Geometric Origins of the Three-Generation Structure)

本理論における「価値創出」のプロセスは、物理学における素粒子の三世代構造と呼応する、三段階の幾何学的レイヤーによって構成されています。

## 1. 第1世代：空間と安定 (Spatial Stability)
* **対応次元:** $x, y, z$ (直交空間軸)
* **幾何学的起源:** メタン正四面体の静止・均衡状態。
* **定義:** 存在の基盤。力が完全に相殺され「0,0」の原点を形成している静的なマテリアルのフェーズ。

## 2. 第2世代：動態と意志 (Kinetic Dynamics)
* **対応次元:** $u, v, w$ (回転・熱量軸)
* **幾何学的起源:** 四元数（クォータニオン）で定義される回転エネルギー。
* **定義:** 変化の起点。個人の「したいこと」＝内発的熱量（角速度 $\omega$）が静止状態を突破し、系を駆動させる動的なフェーズ。

## 3. 第3世代：共鳴と価値 (Resonant Value)
* **対応次元:** $r, i, j$ (虚数・副虚数次元)
* **幾何学的起源:** テンソル積 $\otimes$ による他者次元との結合。
* **定義:** 価値の実体化。自己の熱量が他者の精神的次元と共鳴し、「支える」という相互作用を通じて、新たな「0,0（次の問い）」を生成する進化のフェーズ。
